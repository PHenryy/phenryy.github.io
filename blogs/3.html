  1. 安装 tailwindcss
npm install tailwindcss
  1. 创建全局 css (或其他 css 预处理器) 文件，并加入以下代码
@tailwind base;

@tailwind components;

@tailwind utilities;

在 main.js 中引入

  1. 执行命令创建 tailwindcss 配置文件 (自定义主题，插件等)
npx tailwind init
  1. 配置 postcss.config.js
module.exports = {
  plugins: [require('tailwindcss')('tailwind.config.js'), require('autoprefixer')()]
};
